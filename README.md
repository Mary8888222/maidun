# 麦盾 · 小麦孢子病害智能监测平台

面向“基于机器人与 DNA-PCR 技术的孢子病害识别关键技术研究”的业务系统。平台覆盖电子地图、大数据驾驶舱、病害档案、监控视频、识别监测、结果分析、防控管理、信息管理和机器人中心。

## 运行

要求 Node.js 22.5 或更高版本。

```powershell
npm start
```

打开 `http://localhost:5173`。

- 账号：`tqx`
- 密码：`12345678`

数据库首次启动时自动创建在 `database/maidun.db`，表结构见 `database/schema.sql`。重置系统初始数据：

```powershell
npm run db:reset
```

## 数据库

系统包含 10 张业务表：用户、试验田、设备、机器人任务、PCR 样本、识别结果、风险预警、防控措施、环境记录和视频通道。内置以四川省营山县西桥镇为主的 6 个示范田、7 台设备、42 份 PCR 样本、30 条识别结果、180 条环境记录及完整预警防控闭环数据。检测员包括唐琦翔、陈婉、何佳萍、蒙璐、彭慧琼、唐婉婷、肖闵月、赵单。

当前版本使用 Node.js 内置 SQLite，无第三方运行依赖，可直接部署并进行二次开发。

## 地图与视频素材

- 电子地图使用 OpenStreetMap 数据与瓦片，地图数据版权归 OpenStreetMap contributors。
- 四路麦田视频来自 Pexels：13423170、32505199、33421355、32487386，已压缩为网页播放规格并存放在 `public/media`。
